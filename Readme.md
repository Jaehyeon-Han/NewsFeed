# 뉴스피드 프로젝트
NewsFeed는 회원가입, 로그인, 게시글 및 댓글 기능을 제공하는 REST API기반 프로젝트입니다.
JWT와 로그인 필터를 통해 보안성을 강화하고, 팔로우 및 피드 간 사용자 연결 기능을 구현했습니다.


---


## 기술스택
- Java 17
- Spring Boot 3.4.6
- Spring Data JPA
- MySQL
- Gradle 8.14(wrapper)

## 주요 기능

- User CRUD
  - 회원가입
  - 로그인
  - 회원 정보 조회, 수정, 탈퇴
  - 비밀번호 변경
- Post CRUD
  - 전체 및 단일 게시글 조회
  - 게시글 작성, 수정, 삭제
- Comment
  - 댓글 작성
  - 댓글 조회
  - 댓글 수정, 삭제
- Follow
  - 팔로우 요청
  - 언팔로우
  - 팔로잉 목록
  - 팔로워 목록
